A simple personal portfolio website built using HTML to showcase my profile, skills, and projects. The website helps recruiters quickly understand my work and access my GitHub projects.

## Features
- Personal introduction section
- Projects showcase
- GitHub profile link
- Simple and responsive design
- Easy navigation

## Technologies Used

- CSS (optional)
- Basic web development concepts

## How to Use
1. Download or clone the repository.
2. Open `Portfolio.html` in any web browser.
3. Navigate through the portfolio sections to view projects and information.
